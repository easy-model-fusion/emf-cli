# Easy Model Fusion Project

## Project setup
```
emf-cli install
```

### Add a model
```
emf-cli model add <model-name>
```

### Tidy up the project
```
emf-cli tidy
```

### Build the project
```
emf-cli build
```

### Customize configuration
See [Configuration Reference](https://easy-model-fusion.github.io/docs/).
